# Ecommerce Sales Analysis

This project presents a detailed analysis of ecommerce sales data using **Microsoft Excel**.  
It aims to identify key insights such as sales trends, top-performing products, and customer purchase behavior — enabling data-driven business decisions.

---

# Project Overview

The **Ecommerce Sales Analysis** project focuses on cleaning, organizing, and analyzing sales data from an online store.  
Through Excel’s built-in tools and visualizations, it reveals important performance metrics that can help improve marketing, inventory, and revenue strategies.

---

# File Included

- **Ecommerce Sales Analysis.xlsx** → Main Excel workbook containing:
  - Raw data (sales transactions)
  - Cleaned data sheet
  - Pivot tables and charts
  - Interactive dashboard summarizing key KPIs

---

# Key Insights

- Monthly and yearly **sales trends**
- **Top-selling products** and categories
- **Revenue and profit margin** calculations
- **Customer segmentation** based on purchase frequency
- **Regional sales performance** comparison

---

# Skills Demonstrated

- Data cleaning and transformation in Excel  
- Use of advanced formulas: `SUMIF`, `VLOOKUP`, `IF`, etc.  
- Pivot Tables and Pivot Charts  
- Dashboard design and storytelling with visuals  
- Data-driven decision making  

---

# Tools Used

- **Microsoft Excel** (main analysis)  
- **Charts & Pivot Tables** (visualization)  
- **Conditional Formatting** (highlighting key trends)
